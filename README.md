
WorkMatch
=========

Versi: sederhana, Swing GUI, tanpa database (in-memory ArrayList).
Login sample: admin / 123

Cara import ke NetBeans 27 + JDK 25:
1. File -> Open Project -> pilih folder WorkMatch (yang berisi pom.xml)
2. Clean and Build -> Run Project

Fitur:
- Login
- CRUD Lowongan (Judul, Perusahaan, Lokasi, Deskripsi)
